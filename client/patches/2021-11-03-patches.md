## 必须补丁（Critical）

这些补丁直接影响客户端稳定性，**必须全部打上**：

| 编号 | 名称 | 说明 |
|------|------|------|
| #71 | Ignore Resource Errors | 缺失资源时报警告而非崩溃。**最关键的稳定性补丁** |
| #72 | Ignore Missing Palette Error | 缺失调色板文件时不崩溃 |
| #214 | Restore Model Culling | 修复模型渲染问题，防止渲染相关崩溃 |
| #234 | Ignore Lua Errors | Lua 脚本错误时不崩溃 |

## 补丁操作步骤

1. 下载 NEMO 或 WARP
2. 加载 `2021-11-03_Ragexe_1635824489.exe`
3. 勾选上述所有必须补丁和推荐补丁
4. 点击 Apply / 应用
5. 保存为 `2021-11-03_Ragexe_patched.exe`

## 验证方法

打完补丁后，启动客户端：
- 如果之前崩溃的地图现在只报警告（而非崩溃）→ 补丁生效
- 如果仍然崩溃 → 检查 data.grf 版本是否匹配

## RSW 降级应急方案

如果替换 data.grf 后仍有个别地图崩溃（新版 RSW 格式问题）：

1. 下载 [GRF Editor v1.8.9.7](https://rathena.org/board/files/file/2766-grf-editor/)
2. 打开 data.grf，找到崩溃地图的 `.rsw` / `.gat` / `.gnd` 文件
3. 提取并用 GRF Editor 降级 RSW 版本
4. 将降级后的文件打入 `custom.grf`（利用 data.ini 优先级覆盖）
5. 参考教程：[Map RSW Downgrade Tutorial](https://www.youtube.com/watch?v=A37mVFm2RlM)
